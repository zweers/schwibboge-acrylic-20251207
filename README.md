list of material: 
3mm plywood
3mm acrylic sheet (cast, uncolored)
3mm nuts and bolts
usb connection to ledstrip: https://www.reichelt.de/de/de/shop/produkt/entwicklerboards_-_breakout-board_mit_microusb_2_0-376530
ledstrip 5V, 8mm wide

assembly: build from bottom to top. solder the ledstrip to the pcb and attach on layer 1
layer 2 provides space for ledstrip and pcb
layer 3 and 4 provide a slot for the acrylic Schwibboge shape.
layer 0 (bottom is optional and provides space for the nuts
